# Форма обратной связи

Вёрстка: секция контактов с формой, боковой колонкой с телефоном, email и адресом.

## Возможности

- **Семантическая разметка:** `header`, `main`, `section`, `aside`, подписи к полям, `aria-labelledby` / `aria-live` для доступности.
- **БЭМ** в классах (`contact-form__field`, `contact-info__item` и т.д.).
- **Стили:** CSS-переменные, адаптивная сетка, оформление карточек и формы.
- **Скрипт:** проверка полей перед отправкой, сообщения об ошибках, демонстрационная «отправка» без сервера (см. ниже).
- **Шрифты:** [Manrope](https://fonts.google.com/specimen/Manrope) и [Source Sans 3](https://fonts.google.com/specimen/Source+Sans+3) с Google Fonts.
- **Иконки:** [Material Symbols Outlined](https://fonts.google.com/icons).

## Как открыть

Скачайте папку проекта и откройте `index.html` в браузере (двойной щелчок или через меню «Открыть файл»).


## Форма и бэкенд

Сейчас у формы `action="#"` и отправка перехватывается в JavaScript: при успешной проверке показывается сообщение об успехе, данные на сервер не передаются.

Чтобы подключить реальную отправку:

1. Укажите URL обработчика в атрибуте `action` у `<form>` в `index.html`.
2. При необходимости задайте метод `post` и скрытые поля (например, CSRF-токен).
3. Уберите или измените обработчик `submit` в `js/main.js`, чтобы данные уходили на сервер (или используйте сторонний сервис форм).
