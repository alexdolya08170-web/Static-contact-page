# Лендинг для майстра манікюру

Односторінковий сайт-візитка з можливістю онлайн-запису на процедури.

## Стек технологій

*   React
*   Formik + Yup (валідація форм)
*   Framer Motion (анімації)
*   Axios (робота з API/JSON)
*   CSS Modules

## Функціонал

*   Завантаження даних про послуги та портфоліо через JSON
*   Форма запису з валідацією телефону та імені
*   Адаптивна верстка під мобільні пристрої
*   Плавна навігація по секціях
